## 📝 概念题

### 5.1 ⭐
用**一句话**定义特征值和特征向量：
`Av = λv`（`v ≠ 0`）
- (a) v 是什么？（方向不变的向量）
- (b) λ 是什么？（拉伸倍数）
- (c) 为什么要求 `v ≠ 0`？

### 5.2 ⭐⭐
**几何直觉**：
- 旋转矩阵（非 0°/180°）有实特征向量吗？为什么？
- 单位矩阵 I 的特征值是什么？特征向量呢？（任意非零向量）
- 对角矩阵 `diag(2, 3, 5)` 的特征值和特征向量？

### 5.3 ⭐⭐
**特征值的性质**（判断 + 证明思路）：
- (a) `tr(A) = λ1 + λ2 + ... + λn`（迹 = 特征值之和）
- (b) `det(A) = λ1 · λ2 · ... · λn`（行列式 = 特征值之积）
- (c) A 可逆 ⟺ 所有特征值 ≠ 0
- (d) A 和 A^T 有相同的特征值（但特征向量不同）

### 5.4 ⭐⭐⭐
**对称矩阵的谱定理**（关键！）：
- 实对称矩阵的特征值都是**实数**
- 不同特征值对应的特征向量**正交**
- 可正交对角化：`A = QΛQ^T`，其中 Q 正交

这个定理为什么重要？（PCA、二次型、量子力学都依赖它）

## ✍️ 计算题

### 5.5 ⭐
手算特征值和特征向量：

- (a) `A = [[3, 0], [0, 2]]`
- (b) `A = [[2, 1], [1, 2]]`
- (c) `A = [[4, -2], [1, 1]]`

步骤：
1. 写特征方程 `det(A - λI) = 0`
2. 解出 λ
3. 对每个 λ 解 `(A - λI)v = 0` 得 v

## 💻 编程题

### 5.10 ⭐⭐
用 NumPy 验证上面所有手算结果：

```python
import numpy as np
A = np.array([[2, 1], [1, 2]], dtype=float)
eigenvalues, eigenvectors = np.linalg.eig(A)
# 验证 Av = λv
for i in range(len(eigenvalues)):
    v = eigenvectors[:, i]
    λ = eigenvalues[i]
    assert np.allclose(A @ v, λ * v)
```

### 5.11 ⭐⭐⭐
**幂迭代法（Power Iteration）** 手写求最大特征值：

```python
def power_iteration(A, num_iter=1000, tol=1e-10):
    """
    返回 A 的最大（绝对值）特征值 λ 和对应特征向量 v
    不调用 np.linalg.eig
    """
    n = A.shape[0]
    v = np.random.randn(n)
    v = v / np.linalg.norm(v)
    for _ in range(num_iter):
        v_new = A @ v
        v_new = v_new / np.linalg.norm(v_new)
        if np.linalg.norm(v_new - v) < tol:
            break
        v = v_new
    λ = v @ A @ v  # Rayleigh 商
    return λ, v
```

用 `[[4, 1], [2, 3]]` 测试，对比 `np.linalg.eig`。

### 5.12 ⭐⭐⭐
**可视化**：随机生成 2×2 对称矩阵 A，画出单位圆经过 A 变换后变成椭圆：
- 椭圆的**长/短轴方向** = A 的特征向量
