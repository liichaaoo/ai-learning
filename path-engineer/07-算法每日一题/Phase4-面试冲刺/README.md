# Phase 4 · 面试冲刺（4 周 · 60 题）⭐

> 🎯 **目标**：针对**目标公司**的高频题强化，进入面试状态。
>
> ⏱️ **周期**：4 周（配合主路径阶段 05 简历与面试，**已开始投递**）
>
> 📊 **题量**：60 题（每天 2~3 题，强度上来）

---

## 📌 核心原则

> **这一阶段刷题要"投其所好"。**
>
> 不再按主题刷，而是按**目标公司**刷 CodeTop 上的高频题。

---

## 🛠️ 工具：CodeTop 用法

[https://codetop.cc](https://codetop.cc)

1. 选公司（字节 / 腾讯 / 阿里 / 美团 / Shopee 等）
2. 选岗位（Java / 后端）
3. 按"频率"排序
4. 每家公司刷前 30 题

---

## 🗓️ 4 周计划

### Week 1：目标公司 Top 1（20 题）

> 例：投腾讯 → 刷腾讯高频前 20 题

- [ ] 列出目标公司 Top 1 的高频前 20 题
- [ ] 每题限时 25 分钟
- [ ] 超时直接看题解，归入错题本
- [ ] 周末复盘：哪些主题还是弱项？

### Week 2：目标公司 Top 2 + Top 3（20 题）

- [ ] 第 2 家公司 Top 10
- [ ] 第 3 家公司 Top 10
- [ ] 关注**和 Top 1 重合的题**——这些是行业高频，必拿下

### Week 3：模拟面试（10 题）

> 模拟真实面试节奏。

- [ ] 找朋友 / mock 面试网站做 3 场模拟
- [ ] 每场 60 分钟：1 道算法 + 1 道系统设计 + 八股
- [ ] 算法题要求**边写边讲思路**（出声！）
- [ ] 录音回听，找问题

⭐ 推荐免费 mock 平台：
- LeetCode 周赛
- 牛客 / 力扣模拟面试

### Week 4：临场维持（10 题）

> 已经在面了，不要追新题，维持手感。

- [ ] 每天 1~2 道**已做过的高频题**重温
- [ ] 复习模板手册
- [ ] 面试当天早上：看一眼当天可能问的主题模板
