## 📌 心态

> 算法是"易冷易热"的肌肉。**3 个月不刷 → 完全归零；每周 1 题 → 永远在线。**

---

## 🗓️ 节奏建议

### 周末刷题日（推荐）

- 每个周六早上 30 分钟
- 1 道 LeetCode 周赛题（任选一道 Medium）
- 看下别人题解学技巧

### LeetCode 周赛打卡（进阶）

- 每周日晚上的周赛（10:30 ~ 12:00）
- 不强求做出几道，**重点是保持比赛节奏**
- 周赛 Medium 难度通常比面试题略难，做不出来正常

### 偶发触发刷题

什么时候必须重新捡起来？

- 公司内部有人离职 → 老板谈话有暗示 → 立刻进入 Phase 4 模式
- 行业有大动作（裁员潮 / 风口起来）→ 提前准备
- 距离上次 mock 面试超过 6 个月
