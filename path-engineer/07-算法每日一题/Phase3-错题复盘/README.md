# Phase 3 · 错题复盘（4 周）

> 🎯 **目标**：把 Phase 1+2 攒下的错题啃透，把模板内化成肌肉记忆。
>
> ⏱️ **周期**：4 周（配合主路径阶段 04 Agent 与工程化）
>
> 📊 **题量**：错题 × 2 复刷 + 模板梳理（不增加新题）

---

## 📌 核心原则

> **这一阶段不刷新题。** 老老实实回炉重造，效果远好于继续追新。

---

## 🗓️ 4 周计划

### Week 1：错题第一轮重刷

- [ ] 翻 [`../错题本.md`](../错题本.md)，按主题分组
- [ ] 每天 2 题，**不许看之前写的代码**，独立再写一遍
- [ ] 重刷后状态：
  - ✅ 独立 AC：标记"已掌握"
  - 🟡 提示后 AC：留到 Week 3 再刷
  - 📖 还是看题解：列入 Week 4 重点

### Week 2：模板系统化

把 [`../模板/`](../模板/) 下零散的模板整理成一个 **《面试算法模板手册.md》**，要求：

- 每个模板配 1 个最经典例题
- 每个模板有"什么时候用 / 什么时候不用"
- 用 Java 写，符合面试习惯

模板清单（缺哪个补哪个）：

- [ ] 滑动窗口模板（定长 / 不定长）
- [ ] 二分查找四种写法（找等于 / 找左边界 / 找右边界 / 找峰值）
- [ ] BFS 层序遍历模板
- [ ] DFS 回溯三件套
- [ ] 二叉树 DFS 四件套（前中后 + 分治）
- [ ] DP 五步法
- [ ] 背包问题模板（01 / 完全 / 多重）
- [ ] 单调栈模板
- [ ] 链表反转技巧（头插 / dummy 节点 / 三指针）
- [ ] 并查集模板

### Week 3：错题第二轮重刷 + 易错点专题

- [ ] 第一轮没 AC 的错题再来一遍
- [ ] **易错点小专题**：
  - 边界（数组越界、链表 null、二分 mid 溢出）
  - 整数溢出（int vs long）
  - 字符串 charAt vs codePoint
  - HashMap 默认值（getOrDefault）
  - 引用 vs 值传递

### Week 4：白板模拟

> 面试时大概率不让你用 IDE，要在共享文档/白板上手写。

- [ ] 选 10 道高频题，**不打开 IDE，在 Markdown 里手写**
- [ ] 写完贴回 LeetCode 验证
- [ ] 记录哪些地方手写时容易写错（常见：变量名拼错、缩进、分号）

推荐 10 题：
1. 反转链表（迭代+递归两版）
2. LRU 缓存
3. 二叉树层序遍历
4. 最近公共祖先
5. 全排列
6. 三数之和
7. 最长无重复子串
8. 最长递增子序列
9. 编辑距离
10. 接雨水
