每个模板文件结构：

```markdown
# XX 模板

## 1. 适用场景
（什么时候用，3 句话以内）

## 2. 不适用场景
（容易误用的地方）

## 3. 核心代码（Java）
\`\`\`java
// 5~20 行，不要太长
\`\`\`

## 4. 经典例题
- LeetCode XXX. 题目名
- LeetCode XXX. 题目名

## 5. 易错点
（个人血泪史）
```

---

## 💡 写模板的诀窍

1. **用自己的话写**：别复制题解，要内化
2. **越短越好**：能 10 行不写 20 行
3. **用 Java**：保持母语优势，面试就是用 Java 写
4. **配 1 道经典题**：模板不能脱离题
5. **写完隔一周再看**：还能看懂才算成功
