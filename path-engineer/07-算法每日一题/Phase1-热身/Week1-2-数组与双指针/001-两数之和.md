# 1. 两数之和

> 🔗 [LeetCode 链接](https://leetcode.cn/problems/two-sum/) · `Easy` · `哈希`
>
> 📅 2026-05-19 · ✅ 独立 AC · ⏱ 15 min

---

## 题目

给定一个整数数组 `nums` 和一个整数 `target`，在数组中找出**和为 target** 的两个整数，并返回它们的**下标**。

- 同一个元素不能使用两次
- 答案唯一，顺序任意

**示例**：`nums = [2, 7, 11, 15], target = 9` → `[0, 1]`

---

## 思路

**核心**：边遍历边查。对每个 `nums[i]`，去哈希表里找 `target - nums[i]`。

- 哈希表存 `值 → 下标`
- **先查再放**：避免 `nums = [3, 3], target = 6` 时自己配自己

---

## 解法

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int need = target - nums[i];
            if (map.containsKey(need)) {
                return new int[]{map.get(need), i};
            }
            map.put(nums[i], i);
        }
        return new int[0];
    }
}
```

**复杂度**：时间 O(n) · 空间 O(n)
