## 1. 关键接口

```java
@RestController
@RequestMapping("/api/sessions")
@RequiredArgsConstructor
public class ChatController {

    private final ChatSessionService sessionService;
    private final RagService ragService;
    private final ChatMessageService messageService;

    @PostMapping
    public ChatSessionDto create(@AuthUser User user, @RequestBody Map<String, String> body) {
        return sessionService.create(user.id(), user.tenantId(), body.get("title"));
    }

    @PostMapping(value = "/{sid}/chat/stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<ServerSentEvent<String>> chatStream(
            @AuthUser User user,
            @PathVariable Long sid,
            @RequestBody Map<String, String> body) {

        String question = body.get("q");

        // ① 持久化 user message
        messageService.appendUser(sid, question);

        // ② 调 RAG（带 tenant 过滤）
        StringBuilder fullAnswer = new StringBuilder();
        var stream = ragService.askStream(question, user.tenantId());

        // ③ 流式输出 + 末尾再持久化 assistant + citations
        return stream
                .doOnNext(sse -> {
                    if ("answer".equals(sse.event())) fullAnswer.append(sse.data());
                })
                .concatWith(Mono.fromRunnable(() ->
                        messageService.appendAssistant(sid, fullAnswer.toString(), null)));
    }
}
```

## 2. 多租户过滤（强制注入）

> 这是简历亮点 —— 用 metadata + Spring AI 的 filter expression。

```java
public Flux<ServerSentEvent<String>> askStream(String q, String tenantId) {
    var req = SearchRequest.query(q)
            .withTopK(topK)
            .withSimilarityThreshold(threshold)
            .withFilterExpression("tenant_id == '" + tenantId + "'");   // ⭐ 强制

    List<Document> docs = vectorStore.similaritySearch(req);
    // ...
}
```

> ⚠️ **永远不要让 tenantId 来自 request body**——必须从 JWT/SecurityContext 取。否则就是越权漏洞。

## 3. 答案带"可点击的来源"

`citations` 事件返回：

```json
{
  "event": "citations",
  "data": [
    {"id":"c1","source":"晋升手册.pdf","page":"3","snippet":"晋升流程分三步..."},
    {"id":"c2","source":"晋升手册.pdf","page":"5","snippet":"..."}
  ]
}
```

前端处理：

```js
// 监听 SSE
const evt = new EventSource(`/api/sessions/${sid}/chat/stream`);
evt.addEventListener("answer", e => appendText(e.data));
evt.addEventListener("citations", e => renderCitations(JSON.parse(e.data)));
```
