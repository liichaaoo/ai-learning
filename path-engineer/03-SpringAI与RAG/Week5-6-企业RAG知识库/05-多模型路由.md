## 1. 路由策略矩阵 ⭐

| 场景 | 选谁 | 理由 |
|------|------|------|
| 简单 FAQ / 摘要 | qwen-plus（便宜）| 成本敏感 |
| 复杂推理 / 代码生成 | qwen-max / gpt-4o | 质量优先 |
| 国密/隐私场景 | Ollama 本地 llama3 | 不出域 |
| 多语言翻译 | gpt-4o（英文最强）| 质量 |
| 全员/批量 | qwen-turbo（白菜价）| 极致便宜 |

## 3. ModelRouter 实现

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class ModelRouter {

    @Qualifier("qwenPlus") private final ChatModel qwenPlus;
    @Qualifier("qwenMax")  private final ChatModel qwenMax;
    @Qualifier("ollama")   private final ChatModel ollama;

    public ChatModel pick(RouteContext ctx) {
        // 显式指定（前端可让用户挑）
        if (ctx.preferredModel() != null) return byName(ctx.preferredModel());

        // 隐私场景
        if (ctx.tenant().isPrivacyMode()) return ollama;

        // 复杂任务
        if (ctx.taskType() == TaskType.CODE_GEN ||
            ctx.taskType() == TaskType.COMPLEX_REASONING) {
            return qwenMax;
        }

        // 默认：便宜
        return qwenPlus;
    }

    /** 带降级的安全调用 */
    public ChatResponse callWithFallback(Prompt prompt, RouteContext ctx) {
        ChatModel primary = pick(ctx);
        try {
            return primary.call(prompt);
        } catch (Exception e) {
            log.warn("primary {} failed, fallback to qwenPlus: {}", primary.getClass().getSimpleName(), e.getMessage());
            return qwenPlus.call(prompt);
        }
    }

    public record RouteContext(TaskType taskType, Tenant tenant, String preferredModel) {}
}
```
