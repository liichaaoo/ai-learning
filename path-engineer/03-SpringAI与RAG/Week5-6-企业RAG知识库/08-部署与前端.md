## 1. Dockerfile（多阶段）

```dockerfile
# 阶段 1：构建
FROM maven:3.9-eclipse-temurin-21 AS build
WORKDIR /app
COPY pom.xml .
RUN mvn dependency:go-offline -B
COPY src ./src
RUN mvn clean package -DskipTests -B

# 阶段 2：运行
FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY --from=build /app/target/*.jar app.jar
EXPOSE 8080
ENV JAVA_OPTS="-Xms512m -Xmx2g"
ENTRYPOINT ["sh", "-c", "java $JAVA_OPTS -jar app.jar"]
```

## 2. docker-compose.yml（生产级雏形）

```yaml
version: "3.8"
services:
  app:
    build: .
    image: rag-kb:latest
    container_name: rag-kb-app
    depends_on: [mysql, redis, rabbitmq, milvus]
    environment:
      SPRING_PROFILES_ACTIVE: prod
      DASHSCOPE_API_KEY: ${DASHSCOPE_API_KEY}
      SPRING_DATASOURCE_URL: jdbc:mysql://mysql:3306/rag_kb?useSSL=false
      SPRING_REDIS_HOST: redis
      SPRING_RABBITMQ_HOST: rabbitmq
      MILVUS_HOST: milvus
    ports: ["8080:8080"]

  mysql:
    image: mysql:8.4
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: rag_kb
    volumes: ["./volumes/mysql:/var/lib/mysql"]
    healthcheck:
      test: ["CMD", "mysqladmin", "ping", "-h", "localhost"]
      interval: 10s

  redis:
    image: redis:7
    volumes: ["./volumes/redis:/data"]

  rabbitmq:
    image: rabbitmq:3-management

  # 复用 Week 3 的 milvus 三件套
  etcd: { ... }
  minio: { ... }
  milvus: { ... }
```

## 3. 简单前端（任选）

| 路线 | 推荐度 | 备注 |
|------|--------|------|
| **Vue 3 + Vite + Element Plus** ⭐ | 中文项目首选 | 30 分钟搭框架 |
| React + Ant Design | 简历友好 | 同上 |
| 单文件 HTML（fetch + EventSource）| **最快** | **MVP 推荐** |
| Postman/Insomnia 截图 | 凑合也行 | 时间紧时 |

### 单文件 HTML MVP（30 分钟搞定）

```html
<!doctype html>
<html><head><meta charset="utf-8"><title>RAG KB Demo</title>
<style>
  body { font-family: -apple-system, sans-serif; max-width: 800px; margin: 2em auto; }
  .msg { padding: 8px 12px; margin: 4px 0; border-radius: 8px; }
  .user { background: #e3f2fd; text-align: right; }
  .ai { background: #f5f5f5; }
  .cite { color: #666; font-size: 0.9em; margin-top: 4px; }
</style>
</head><body>
<h2>企业知识库</h2>
<div id="messages"></div>
<input id="q" placeholder="提问..." style="width: 70%; padding: 8px;"/>
<button onclick="ask()">问</button>

<script>
const MSG = document.getElementById('messages');
function add(role, text) {
  const div = document.createElement('div');
  div.className = 'msg ' + role;
  div.textContent = text;
  MSG.appendChild(div);
  return div;
}

async function ask() {
  const q = document.getElementById('q').value;
  if (!q) return;
  add('user', q);
  const aiBox = add('ai', '');
  document.getElementById('q').value = '';

  const sid = window.sid || (window.sid = 1);
  const evt = new EventSource(
    `/api/sessions/${sid}/chat/stream?q=${encodeURIComponent(q)}`
  );
  evt.addEventListener('answer', e => { aiBox.textContent += e.data; });
  evt.addEventListener('citations', e => {
    const cites = JSON.parse(e.data);
    const c = document.createElement('div'); c.className = 'cite';
    c.textContent = '来源：' + cites.map(x => `${x.source} 第${x.page}页`).join('；');
    aiBox.appendChild(c);
    evt.close();
  });
}
</script>
</body></html>
```

> 把它放到 `src/main/resources/static/index.html`，访问 `http://localhost:8080/` 直接能用。
