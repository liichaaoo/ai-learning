## 1. 流程图

```
用户 POST /docs                    PENDING（即时返回 docId）
       │
       ▼
RabbitMQ ← 入队 IngestionTask
       │
       ▼
IngestionWorker（消费者）
   ① download from OSS
   ② Tika 解析 → text + metadata
   ③ RecursiveCharacterSplitter 分片  PROCESSING（更新 chunks=N, processed=0）
   ④ 批量 embed + 批量写 Milvus
       每批回写 processed += batch_size
   ⑤ 持久化 chunk 到 MySQL          READY（end）
失败 → 写 error_message              FAILED
```

## 2. 关键代码

### 2.1 上传接口（异步）

```java
@PostMapping("/docs")
public Map<String, Object> upload(@RequestParam MultipartFile file,
                                   @AuthUser User user) throws IOException {
    String sha = sha256(file.getInputStream());
    Optional<Document> exists = docRepo.findByTenantIdAndSha256(user.tenantId(), sha);
    if (exists.isPresent()) return Map.of("docId", exists.get().getId(), "deduplicated", true);

    String objectKey = ossService.upload(file);
    Document doc = docRepo.save(Document.builder()
            .userId(user.id()).tenantId(user.tenantId())
            .filename(file.getOriginalFilename()).sha256(sha)
            .status("PENDING").build());

    rabbit.convertAndSend("rag.ingest", new IngestTask(doc.getId(), objectKey));
    return Map.of("docId", doc.getId(), "status", "PENDING");
}
```

### 2.2 IngestionWorker（消费者）

```java
@Component
@RequiredArgsConstructor
public class IngestionWorker {
    private final DocumentRepository docRepo;
    private final ChunkRepository chunkRepo;
    private final VectorStore vectorStore;
    private final OssService oss;
    private final TikaParser tika;
    private final RecursiveCharacterSplitter splitter;

    @RabbitListener(queues = "rag.ingest")
    @Transactional
    public void handle(IngestTask task) {
        Document doc = docRepo.findById(task.docId()).orElseThrow();
        try {
            doc.setStatus("PROCESSING"); docRepo.save(doc);

            byte[] data = oss.download(task.objectKey());
            var parsed = tika.parse(data);

            List<String> chunks = splitter.split(parsed.text());
            doc.setTotalChunks(chunks.size());
            docRepo.save(doc);

            // 批量 embed + 写库
            int batchSize = 16;
            for (int i = 0; i < chunks.size(); i += batchSize) {
                List<String> batch = chunks.subList(i, Math.min(i + batchSize, chunks.size()));
                List<org.springframework.ai.document.Document> docs = batch.stream()
                        .map(c -> new org.springframework.ai.document.Document(c, Map.of(
                                "doc_id", doc.getId(),
                                "tenant_id", doc.getTenantId(),
                                "source", doc.getFilename())))
                        .toList();
                vectorStore.add(docs);
                // 持久化 chunk → MySQL（保留与 vector_id 的关系）
                for (int k = 0; k < docs.size(); k++) {
                    chunkRepo.save(Chunk.builder()
                            .documentId(doc.getId())
                            .vectorId(docs.get(k).getId())
                            .chunkSeq(i + k)
                            .content(batch.get(k))
                            .build());
                }
            }
            doc.setStatus("READY"); docRepo.save(doc);
        } catch (Exception e) {
            doc.setStatus("FAILED");
            doc.setErrorMessage(e.getMessage());
            docRepo.save(doc);
            throw e;
        }
    }
}
```

### 2.3 进度查询

```java
@GetMapping("/docs/{id}/progress")
public Map<String, Object> progress(@PathVariable Long id) {
    Document d = docRepo.findById(id).orElseThrow();
    long persisted = chunkRepo.countByDocumentId(id);
    return Map.of(
            "status", d.getStatus(),
            "totalChunks", d.getTotalChunks(),
            "processedChunks", persisted,
            "errorMessage", d.getErrorMessage());
}
```

## 3. 验收标准

- [ ] 上传一个 50 页 PDF，接口立刻返回（非阻塞）
