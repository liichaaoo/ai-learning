## 1. 用 Spring AI 内置 Memory（最简）

```java
@Bean
public ChatMemory chatMemory() {
    return new InMemoryChatMemory();   // 入门用，生产换 RedisChatMemory
}

@Service
public class RagService {

    private final ChatClient chatClient;

    public RagService(ChatClient.Builder b, VectorStore vs, ChatMemory memory) {
        this.chatClient = b
                .defaultAdvisors(
                        new MessageChatMemoryAdvisor(memory),
                        new QuestionAnswerAdvisor(vs))
                .build();
    }

    public Flux<String> askStream(String conversationId, String q) {
        return chatClient.prompt()
                .advisors(spec -> spec.param(
                        AbstractChatMemoryAdvisor.CHAT_MEMORY_CONVERSATION_ID_KEY, conversationId))
                .user(q)
                .stream().content();
    }
}
```

## 2. 自己实现一个 RedisChatMemory

```java
@Component
@RequiredArgsConstructor
public class RedisChatMemory implements ChatMemory {

    private final StringRedisTemplate redis;
    private final ObjectMapper om = new ObjectMapper();

    @Override
    public void add(String conversationId, List<Message> messages) {
        String key = key(conversationId);
        for (Message m : messages) {
            redis.opsForList().rightPush(key, serialize(m));
        }
        redis.opsForList().trim(key, -100, -1);     // 只保留最近 100 条
        redis.expire(key, Duration.ofDays(7));
    }

    @Override
    public List<Message> get(String conversationId, int lastN) {
        Long size = redis.opsForList().size(key(conversationId));
        if (size == null || size == 0) return List.of();
        long start = Math.max(0, size - lastN);
        List<String> raw = redis.opsForList().range(key(conversationId), start, -1);
        return raw == null ? List.of() :
                raw.stream().map(this::deserialize).toList();
    }

    @Override
    public void clear(String conversationId) {
        redis.delete(key(conversationId));
    }

    private String key(String id) { return "rag:chat:" + id; }
    private String serialize(Message m) { /* JSON */ ... }
    private Message deserialize(String s) { /* JSON */ ... }
}
```

## 3. 上下文压缩（高级）

> 当历史对话很长时，**别全塞进 prompt**——总结成一两句话。

```java
public List<Message> compressIfNeeded(List<Message> history) {
    int totalTokens = history.stream().mapToInt(m -> approxTokens(m.getContent())).sum();
    if (totalTokens < 2000) return history;

    String summary = chatClient.prompt()
            .system("把以下对话压缩成 5 句话以内的摘要，保留关键事实。")
            .user(history.stream().map(Message::getContent).collect(Collectors.joining("\n")))
            .call()
            .content();

    // 保留最近 4 条 + 一条 summary
    List<Message> recent = history.subList(Math.max(0, history.size() - 4), history.size());
    return Stream.concat(
            Stream.of(new SystemMessage("【对话摘要】" + summary)),
            recent.stream()
    ).toList();
}
```

## 4. 验收标准
