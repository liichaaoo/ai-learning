## 2. 评测指标

| 指标 | 含义 | 怎么算 |
|------|------|-------|
| **Recall@K** | Top-K 里有没有期望来源 | 命中数 / 总题数 |
| **MRR**（Mean Reciprocal Rank）| 期望来源在第几位（取倒数）| 均值 |
| **答案命中率** | 答案是否包含期望关键词 | 关键词 OR 命中 |
| **拒答正确率** | 真没答案的题，是否拒答 | 准确拒答数 / 总题数 |
| **平均延迟** | P50 / P95 | k6 压测 |

## 3. 自动评测脚本

```java
@Test
void runEvaluation() throws Exception {
    List<TestCase> cases = loadCases("eval/eval-set.yaml");
    EvalReport report = new EvalReport();

    for (TestCase tc : cases) {
        long t0 = System.currentTimeMillis();
        var ans = ragService.ask(tc.q(), null);
        long elapsed = System.currentTimeMillis() - t0;

        boolean kw = tc.keywords().stream().anyMatch(k -> ans.answer().contains(k));
        boolean src = ans.citations().stream()
                .anyMatch(c -> c.source().contains(tc.source()));

        // MRR
        int rank = -1;
        for (int i = 0; i < ans.citations().size(); i++) {
            if (ans.citations().get(i).source().contains(tc.source())) {
                rank = i + 1; break;
            }
        }

        report.add(tc.id(), kw, src, rank, elapsed);
    }

    report.printSummary();
    report.saveCsv("eval/results-" + LocalDate.now() + ".csv");
}
```

输出示例：

```
=== Evaluation Result ===
Total: 30
Source Hit: 27/30 (90.0%)         ← Recall@3
Keyword Hit: 24/30 (80.0%)
MRR: 0.78
P50 latency: 1180ms
P95 latency: 1620ms
```

## 4. 调优实验：扫一遍配置

```java
int[][] configs = {
        // {topK, chunkSize, overlap, threshold, useRerank?(0/1)}
        {3,  500, 50,  0.5, 0},
        {5,  500, 100, 0.5, 0},     // baseline
        {5,  500, 100, 0.5, 1},     // + rerank
        {8,  500, 100, 0.5, 0},
        {5,  300, 30,  0.5, 0},
        {5,  800, 150, 0.5, 0},
        {5,  500, 100, 0.7, 0},     // 严阈值
};
```

写个对比表：

| 配置 | Recall@3 | MRR | P95 延迟 |
|------|---------|-----|---------|
| 纯向量 topK=3 | 65% | 0.55 | 1.1s |
| **纯向量 topK=5** | **80%** | **0.70** | **1.2s** |
| **+Rerank** | **90%** | **0.85** | **1.5s** |
| chunk=300 | 70% | 0.58 | 1.2s |
| chunk=800 | 75% | 0.62 | 1.3s |
| 阈值 0.7 | 75%（拒答 5）| 0.71 | 1.2s |

> 🎯 **简历叙述**：
>
> 「通过分片参数 / Top-K / Reranker 对比实验（基于 30 题人工评测集），
> 把 Top-3 召回率从 65% → 90%，MRR 从 0.55 → 0.85，
> P95 延迟控制在 1.5s 以内。」
