#### 关键叙事原则

把现有 7 年 Java 工作的项目，**重写为"AI 应用平台地基"** 的叙事：

| ❌ 之前的写法 | ✅ AI 时代的写法 |
|--------------|----------------|
| 负责 XX 业务后端开发 | 主导 XX 微服务架构，**为后续 AI 应用提供高可用底座** |
| 优化 SQL，QPS 提升 30% | 优化数据查询性能，**支撑后续 RAG 系统的实时检索** |
| 设计 XX 系统架构 | 设计可扩展架构，**支持灵活接入大模型与 Agent** |

### Week 3：系统设计题

> **5 年+ 的工程师，必考系统设计。这是你 7 年经验的杀手锏。**

#### 必练 5 题

1. **设计一个企业级 RAG 系统**
   - 100W+ 文档、1W+ QPS
   - 多租户、多模型、降级、缓存

2. **设计一个多模型路由网关**
   - GPT-4 / Claude / 通义混合
   - 按场景路由、按成本路由
   - 限流、降级、统一日志

3. **设计一个 AI 客服系统**
   - 高可用方案
   - Knowledge 自动更新
   - 答非所问的兜底

4. **设计一个 Token 计费系统**
   - 多用户、多模型、多功能
   - 实时用量统计 + 告警

5. **设计一个 Agent 编排平台**
   - 类似 Dify / Coze
   - 流程编排、版本管理、监控

📁 整理位置：[`./系统设计/`](./系统设计/)

### Week 4：面试实战

#### 投递渠道（按优先级）

##### 1. 腾讯内部转岗（最优）⭐
- 目标 BG：TEG（混元）/ CSIG（云大模型）/ PCG（内容 AI）/ IEG（游戏 AI）
- 操作：当前部门 → KM 内推 → 内部转岗系统
- 优势：保 P 级 + 保股票

##### 2. 大厂外跳
- 字节、阿里、美团、京东、百度、网易
- 渠道：脉脉 / 内推 / 猎头

##### 3. AI 独角兽
- MiniMax、智谱、月之暗面、零一万物、阶跃星辰、DeepSeek
- 渠道：直接官网投递 + 脉脉
