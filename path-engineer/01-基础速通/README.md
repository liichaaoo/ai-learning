## 🗓️ 3 周计划

### Week 1：Python 速通（3-5 天）

#### 你已经会 Java，所以学 Python 只是语法切换

- [ ] **Day 1**：Python 基础语法（基于 Java 对照学）
  - 变量、数据类型、控制流（`if / for / while`）
  - 函数定义、参数、返回值
  - 类与对象（重点对比 Java OOP）

- [ ] **Day 2**：Python 特色 + 常用数据结构
  - `list` / `dict` / `set` / `tuple`
  - 列表推导式、生成器
  - 切片操作、`*args` / `**kwargs`

- [ ] **Day 3**：Python 进阶
  - 装饰器（`@decorator`）
  - `with` 语句、上下文管理器
  - 异常处理

- [ ] **Day 4**：NumPy 基础
  - `ndarray` 创建与操作
  - 广播机制（理解即可）
  - 矩阵运算

- [ ] **Day 5**：能看懂的目标
  - 找一段 PyTorch / sklearn 代码读懂
  - 能用 Python 写一个简单的脚本

#### 📚 推荐资源
- [廖雪峰 Python 教程](https://www.liaoxuefeng.com/wiki/1016959663602400) 前 30%
- 或《Python Crash Course》前 8 章

#### ❌ 不学（重要！）
- Django / Flask / FastAPI（你已经会 Spring）
- 多线程多进程（不是 AI 重点）
- asyncio 深度用法
- Python 包管理深坑

### Week 3：数学补盲（**只学 LLM 相关**）

#### 你已经学了线性代数（在 path-research/01/），这周补关键缺口

- [ ] **Day 1-2**：线性代数复盘（针对 LLM）
  - 向量内积、范数（**Embedding 相似度要用**）
  - 矩阵乘法（**Attention 要用**）
  - 你已有的笔记：[`path-research/01-数学与编程基础/1.1-线性代数/`](../../path-research/01-数学与编程基础/1.1-线性代数/)

- [ ] **Day 3**：概率与统计（精简版）
  - 条件概率、贝叶斯公式（**RAG 召回要懂**）
  - 概率分布（高斯、伯努利）
  - 期望与方差

- [ ] **Day 4**：微积分（精简版）
  - 导数、偏导
  - 梯度的几何意义
  - **知道反向传播是干嘛**就行（不推导）

- [ ] **Day 5**：信息论（极少量）
  - 交叉熵（**LLM 损失函数**）
  - KL 散度（**模型对齐要用**）

#### ❌ 不学
- 特征值分解 / SVD 的详细推导（已有笔记可参考但不深学）
- 凸优化理论
- 概率统计检验
