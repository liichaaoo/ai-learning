## 一、导数（Derivative）是什么

### 直觉 1：变化率

一个函数 $f(x)$，**x 变一点点，f 变多少？** 这个"变化速率"就是导数。

### 直觉 2：切线斜率

画出 $f(x)$ 的曲线，某一点的**切线斜率**就是该点的导数。

### 符号

$$f'(x) = \frac{df}{dx}$$

### 例子（记住结论，不管推导）

| 函数 $f(x)$ | 导数 $f'(x)$ |
|-----------|------------|
| $x^2$     | $2x$       |
| $x^n$     | $nx^{n-1}$ |
| $e^x$     | $e^x$      |
| $\ln(x)$  | $1/x$      |
| $\sin(x)$ | $\cos(x)$  |

> 💡 **提醒**：**不用背这些**。知道"导数是一个变化率 / 斜率"就够了。PyTorch 会自动算。

### 在 AI 里的意义

```
Loss 函数 L(θ)  关于参数 θ 的导数
 = Loss 随 θ 变化的快慢
 = "微调 θ 一点点，loss 会变多少"
```

**训练 = 调整 θ 让 Loss 变小**。导数告诉我们"该往哪个方向调，调多少"。

## 二、梯度（Gradient）⭐⭐⭐

### 2.1 直觉：多维导数

如果函数有多个输入（神经网络的参数可能有几十亿个！），**每个输入都有一个偏导数**。

把所有偏导数打包成一个向量 → **梯度**。

符号：$\nabla_\theta L$（读作"nabla theta L"）

### 2.2 梯度的"方向"

**梯度方向 = 函数增长最快的方向**

相反地，**梯度的反方向 = 函数下降最快的方向**。

### 2.3 梯度下降（Gradient Descent）

训练神经网络的**核心算法**。

#### 流程

```
1. 初始化参数 θ
2. 重复下面的步骤：
   a) 算 loss: L = f(θ, data)
   b) 算梯度: g = ∇L (loss 对参数的梯度)
   c) 更新参数: θ = θ - η * g   ← 关键！
3. 直到 loss 不再下降
```

其中：
- $\eta$ 叫 **学习率（learning rate）**，是超参数
- **减号**：往梯度的反方向走（让 loss 变小）

#### 视觉化

```
Loss
 │
 │  *     ← 起点（loss 很大）
 │   \
 │    \   ← 每步沿梯度反方向走一小步
 │     \
 │      \__
 │         \___     ← 终点（loss 最小）
 │____________________  参数 θ
```

就像**蒙着眼下山**：
- 只知道脚下"哪边最陡"（梯度）
- 每次往最陡的**反方向**走一小步（反方向 = 下坡）
- 一直走到平地

### 2.4 学习率 η 的玄机

```
η 太大  → 每步跨太大，跳过谷底甚至震荡发散
η 太小  → 每步走太慢，要跑几百万步
η 合适  → 平稳下降
```

**实际工作中**：
- 默认试 `lr=1e-3`（0.001）
- Adam 优化器常用 `lr=3e-4`
- 大模型预训练用 `lr=1e-4` 到 `lr=5e-5`

## 三、链式法则（Chain Rule）⭐⭐

### 3.1 直觉

复合函数求导的规则。

**例子**：$y = f(g(x))$

$$\frac{dy}{dx} = \frac{dy}{dg} \cdot \frac{dg}{dx}$$

**人话翻译**：**导数可以"接力传递"**。

### 3.2 神经网络里的意义

神经网络就是一堆函数"套娃"：

```
x → Layer1 → h1 → Layer2 → h2 → Layer3 → y → Loss
                                          ↓
                             要算 Loss 关于 x 的梯度
                             需要逐层"反向传递"导数
                             这就是反向传播
```

---

## 四、反向传播（Backpropagation）⭐⭐⭐

这是训练神经网络**最核心的算法**。

### 4.1 它做什么

**一个词**：**逐层反向算梯度**。

### 4.2 流程（必懂）

```
正向传播 (Forward):
   输入 → 层1 → 层2 → 层3 → 输出 → Loss

反向传播 (Backward):
   Loss → 层3 梯度 → 层2 梯度 → 层1 梯度 → 输入梯度
   
   按链式法则，梯度从输出端"反向传"到输入端
   每一层的参数梯度都被算出来，用于更新
```

### 4.3 为啥叫"反向"

因为算梯度的顺序和前向正好相反 —— **从输出开始，往输入方向算**。

### 4.4 你只需要知道这些

- ✅ 反向传播 = 链式法则在神经网络上的自动应用
- ✅ PyTorch 的 `loss.backward()` 一行搞定
- ✅ 你**永远不用自己实现**
- ❌ 不需要手推

## 五、PyTorch 的 autograd（**感受魔法**）

### 5.1 你以后只会写这样的代码

```python
import torch

# 1. 定义模型和数据
model = MyModel()
x, y = get_batch()

# 2. 前向传播
y_pred = model(x)
loss = cross_entropy(y_pred, y)

# 3. 反向传播（这一行自动算所有梯度！）
loss.backward()

# 4. 用梯度更新参数
optimizer.step()
optimizer.zero_grad()
```

### 5.2 autograd 是什么

PyTorch 自动求导系统，**记录前向传播每一步的计算**，反向时按链式法则自动算出所有梯度。

```python
import torch

x = torch.tensor(3.0, requires_grad=True)
y = x ** 2 + 2 * x + 1   # y = x² + 2x + 1

y.backward()             # 自动算 dy/dx

print(x.grad)            # tensor(8.)  ← dy/dx = 2x + 2，x=3 时等于 8 ✅
```

**你连公式都不用想**，PyTorch 帮你算好。

## 六、优化器（Optimizer）（认识 3 个即可）

梯度下降只是"**最朴素**"的做法。实际工作中用改进版优化器：

| 优化器 | 核心思想 | 用在哪 |
|-------|---------|------|
| **SGD** (Stochastic GD) | 基础梯度下降 + 随机采样 mini-batch | 图像任务常用 |
| **Adam** | 自适应学习率 + 动量 | **默认选它**（LLM、NLP） |
| **AdamW** | Adam + 更好的权重衰减 | **LLM 训练最常用** |

### 你要记住

```python
# 99% 的情况用这个
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-4)
```

## 七、一个完整的训练循环（你以后天天见）

```python
import torch

# 准备
model = MyModel()
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-4)
criterion = torch.nn.CrossEntropyLoss()

for epoch in range(100):
    for x, y in dataloader:
        # 1. 前向传播
        logits = model(x)
        loss = criterion(logits, y)

        # 2. 清零梯度（上一步的梯度还在，要清掉）
        optimizer.zero_grad()

        # 3. 反向传播（autograd 自动算所有梯度）
        loss.backward()

        # 4. 优化器更新参数
        optimizer.step()

    print(f"Epoch {epoch}: loss = {loss.item():.4f}")
```

**今天结束你的目标**：**能看懂这段循环每一步在做什么**。

## 八、常见误区

### ❌ "我要学到能自己实现反向传播"
→ **不需要**。深度学习框架 10 年前就帮你做了。

### ❌ "梯度消失/爆炸我要能手推"
→ 你**需要知道**：
- 梯度消失 = 反向传到深层时梯度变得很小，参数学不动（Sigmoid 旧锅，现在用 ReLU 缓解）
- 梯度爆炸 = 反向传时梯度越乘越大（用 gradient clipping 解决）
- **不需要**推数学原因。

### ❌ "我要背所有激活函数的导数"
→ 认识几个常见的就行：
- **ReLU**：`f(x) = max(0, x)`，导数是 0 或 1
- **Sigmoid**：输出 0-1，用于二分类
- **Tanh**：输出 -1 到 1
- **Softmax**：输出概率分布（Day 3 学过）
