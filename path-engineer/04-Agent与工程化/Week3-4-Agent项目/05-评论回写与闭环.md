## 1. GitHub Reviews API

```java
@Service
@RequiredArgsConstructor
public class GitHubCommentService {

    @Value("${github.token}") private String token;
    private final WebClient client = WebClient.builder()
            .baseUrl("https://api.github.com").build();

    /** 创建一个 Review（含多条 inline 评论 + 总结）*/
    public void postReview(PrEvent pr, List<ReviewComment> comments, String summary) {
        List<Map<String, Object>> inline = comments.stream()
                .filter(c -> c.line() > 0)
                .map(c -> Map.<String, Object>of(
                        "path", c.filename(),
                        "line", c.line(),
                        "side", "RIGHT",
                        "body", formatComment(c)
                ))
                .toList();

        Map<String, Object> body = Map.of(
                "commit_id", pr.headSha(),
                "body", summary,
                "event", "COMMENT",                       // COMMENT / REQUEST_CHANGES / APPROVE
                "comments", inline
        );

        client.post()
                .uri("/repos/{repo}/pulls/{n}/reviews", pr.repoFullName(), pr.number())
                .header("Authorization", "Bearer " + token)
                .header("Accept", "application/vnd.github+json")
                .bodyValue(body)
                .retrieve()
                .toBodilessEntity()
                .block();
    }

    /** 只发一条 issue comment（用于 summary / 跳过通知）*/
    public void postSummary(PrEvent pr, String message) {
        client.post()
                .uri("/repos/{repo}/issues/{n}/comments", pr.repoFullName(), pr.number())
                .header("Authorization", "Bearer " + token)
                .bodyValue(Map.of("body", message))
                .retrieve()
                .toBodilessEntity()
                .block();
    }

    private String formatComment(ReviewComment c) {
        String icon = switch (c.severity()) {
            case BLOCKER -> "🛑";
            case MAJOR   -> "⚠️";
            case MINOR   -> "💡";
            case INFO    -> "ℹ️";
        };
        StringBuilder sb = new StringBuilder();
        sb.append("**").append(icon).append(" ")
          .append(c.severity()).append(" · ").append(c.category()).append("**\n\n");
        sb.append(c.message()).append("\n");
        if (c.suggestion() != null && !c.suggestion().isBlank()) {
            sb.append("\n```suggestion\n").append(c.suggestion()).append("\n```\n");
        }
        sb.append("\n_— by 🤖 AI Code Reviewer_");
        return sb.toString();
    }
}
```

> 💡 **`suggestion` 代码块是 GitHub 特性**：开发者可以一键 commit 你的建议。

## 2. Summary 模板

```java
public String buildSummary(List<ReviewComment> comments, long elapsedMs) {
    Map<Severity, Long> bySev = comments.stream()
            .collect(Collectors.groupingBy(ReviewComment::severity, Collectors.counting()));

    StringBuilder sb = new StringBuilder("## 🤖 AI Code Review 报告\n\n");
    sb.append("| 严重程度 | 数量 |\n|---|---|\n");
    for (Severity s : Severity.values()) {
        long cnt = bySev.getOrDefault(s, 0L);
        if (cnt > 0) sb.append("| ").append(s).append(" | ").append(cnt).append(" |\n");
    }
    sb.append("\n**耗时**：").append(String.format("%.1fs", elapsedMs / 1000.0)).append("\n");

    if (bySev.containsKey(Severity.BLOCKER)) {
        sb.append("\n> 🛑 含 **BLOCKER** 级问题，建议合并前修复。\n");
    } else if (comments.isEmpty()) {
        sb.append("\n> ✅ **未发现明显问题**。\n");
    }

    sb.append("\n<details><summary>审查范围</summary>\n\n");
    sb.append("- 语法：NPE / 异常处理 / 资源释放\n");
    sb.append("- 安全：注入 / 越权 / 敏感信息\n");
    sb.append("- 性能：N+1 / 循环 IO / 锁\n");
    sb.append("</details>\n");
    return sb.toString();
}
```

## 3. 防止重复评论（增量审查）

PR 多次 push 时，避免一直加重复评论：

```java
// 拉取已有评论
public Set<String> fetchExistingCommentKeys(String repo, long prNum) {
    JsonNode reviews = client.get()
            .uri("/repos/{r}/pulls/{n}/reviews", repo, prNum)
            .header("Authorization", "Bearer " + token)
            .retrieve().bodyToMono(JsonNode.class).block();

    Set<String> keys = new HashSet<>();
    for (JsonNode r : reviews) {
        if (!"🤖".equals(safeBody(r).substring(0, 1))) continue;   // 只看 AI 评论
        // ... 解析 inline comments，取 file+line 作为 key
    }
    return keys;
}
```

新评论生成时：

```java
Set<String> existing = fetchExistingCommentKeys(...);
List<ReviewComment> fresh = newComments.stream()
        .filter(c -> !existing.contains(c.filename() + "#" + c.line()))
        .toList();
```
