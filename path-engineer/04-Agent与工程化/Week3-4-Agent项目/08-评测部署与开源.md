## 2. 评测脚本

```java
@SpringBootTest
class EvaluationTest {

    @Autowired ReviewOrchestrator orchestrator;

    @Test
    void runEval() throws IOException {
        List<TestCase> cases = loadTestSet("eval/pr-test-set");
        EvalReport report = new EvalReport();

        for (TestCase tc : cases) {
            long t0 = System.currentTimeMillis();
            var comments = orchestrator.reviewHunk(tc.filename(), tc.diff());
            long elapsed = System.currentTimeMillis() - t0;

            boolean hit = tc.expected().stream().allMatch(exp ->
                comments.stream().anyMatch(c ->
                    c.severity() == exp.severity() &&
                    c.category() == exp.category() &&
                    exp.keywords().stream().anyMatch(k ->
                        c.message().toLowerCase().contains(k.toLowerCase()))));

            report.add(tc.id(), hit, comments.size(), elapsed);
        }

        report.print();
        report.saveCsv("eval/results-" + LocalDate.now() + ".csv");
    }
}
```

期望输出：

```
=== Evaluation ===
Total cases: 30
True positives: 26 (86.7%)              ← 简历主数字
False positives: ~ 4.3 per PR avg       ← 误报率
Avg latency: 23.5s
```

> 🎯 **简历叙述**：「基于 30 个真实 PR 评测，**问题识别率 86.7%**，平均耗时 < 30s」。

## 3. Dockerfile

```dockerfile
# 多阶段
FROM maven:3.9-eclipse-temurin-21 AS build
WORKDIR /app
COPY pom.xml .
RUN mvn dependency:go-offline -B
COPY src ./src
RUN mvn clean package -DskipTests -B

FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY --from=build /app/target/*.jar app.jar
EXPOSE 8080
ENV JAVA_OPTS="-Xms512m -Xmx2g"
ENTRYPOINT ["sh", "-c", "java $JAVA_OPTS -jar app.jar"]
```

## 4. docker-compose.yml

```yaml
services:
  app:
    build: .
    image: ai-code-reviewer:latest
    environment:
      DASHSCOPE_API_KEY: ${DASHSCOPE_API_KEY}
      GITHUB_TOKEN: ${GITHUB_TOKEN}
      GITHUB_WEBHOOK_SECRET: ${GITHUB_WEBHOOK_SECRET}
      SPRING_DATA_REDIS_HOST: redis
    depends_on: [redis]
    ports: ["8080:8080"]

  redis:
    image: redis:7
    volumes: ["./volumes/redis:/data"]

  prometheus:
    image: prom/prometheus
    ports: ["9090:9090"]
    volumes: ["./prometheus.yml:/etc/prometheus/prometheus.yml"]

  grafana:
    image: grafana/grafana
    ports: ["3000:3000"]
```
