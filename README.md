## 使用约定

### 文件命名
- 笔记：`YYYY-MM-DD-主题.md`，如 `2026-05-07-attention机制.md`
- 论文笔记：`论文标题简写-年份.md`，如 `transformer-2017.md`
- 代码项目：英文小写中划线，如 `nano-gpt-reproduce/`

### 学习节奏建议
| 频率 | 动作 |
|------|------|
| 每天 | 写代码 ≥ 30 分钟，更新当日笔记 |
| 每周 | 总结一篇周报到 `00-学习总览/周报/` |
| 每阶段 | 完成至少一个项目 → `产出/` |

### 笔记原则
1. **用自己的话写**，不要复制粘贴
2. **图 + 公式 + 代码** 三位一体
3. 每个笔记结尾留 **3 个还没搞懂的问题**
4. 一周后回头看，能解释清楚才算真懂
