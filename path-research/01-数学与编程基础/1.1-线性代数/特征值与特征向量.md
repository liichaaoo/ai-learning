## 一、最直观的理解：方向不变的轴

### 一句话定义

对方阵 \(A \in \mathbb{R}^{n \times n}\)，若存在**非零向量** \(\boldsymbol{v}\) 和标量 \(\lambda\) 满足：
\[
\boxed{A \boldsymbol{v} = \lambda \boldsymbol{v}}
\]
则 \(\boldsymbol{v}\) 是 \(A\) 的**特征向量**，\(\lambda\) 是对应的**特征值**。

### 几何含义（必须先记住的画面）

把矩阵 \(A\) 看成一个**线性变换**——它把空间扭来扭去。
- 大多数向量被 \(A\) 作用后，**方向会改变**。
- 只有**极少数特殊方向**上的向量，被 \(A\) 作用后**方向不变**，只是被**拉伸或压缩**了 \(\lambda\) 倍。
- 这些"方向不变的轴" = **特征向量**；拉伸的倍数 = **特征值**。

```
        变换前              变换后 (Ax)
          │                     │
   ───────┼───────  →    ───────┼───────
          │                     │  ↗ 蓝色：普通向量，方向改了
                                │
   ↑ 红色：特征向量              ↑ 红色：方向没变，只是变长了 λ 倍
```

### 为什么要求 \(\boldsymbol{v} \neq 0\)？
\(A \cdot \boldsymbol{0} = \lambda \cdot \boldsymbol{0}\) 对**任何** \(\lambda\) 都成立，零向量没有"方向"，无意义。

### \(\lambda\) 的符号告诉你什么

| \(\lambda\) | 几何效果 |
|------------|---------|
| \(\lambda > 1\) | 同向**拉长** |
| \(0 < \lambda < 1\) | 同向**压缩** |
| \(\lambda = 1\) | **不变**（属于"不动方向"） |
| \(\lambda = 0\) | 被压成 0（落入零空间） |
| \(\lambda < 0\) | **反向** + 拉伸 |
| 复数 \(\lambda\) | 包含**旋转**成分（无实特征向量） |

## 二、怎么求？（特征方程的来龙去脉）

### 推导

由 \(A \boldsymbol{v} = \lambda \boldsymbol{v}\)：
\[
A \boldsymbol{v} - \lambda \boldsymbol{v} = 0 \quad\Longrightarrow\quad (A - \lambda I)\boldsymbol{v} = 0
\]

要让方程有**非零解** \(\boldsymbol{v}\)，矩阵 \((A - \lambda I)\) 必须**奇异**（不可逆）：

\[
\boxed{\det(A - \lambda I) = 0}
\]

这就是**特征方程（characteristic equation）**，左边是关于 \(\lambda\) 的 \(n\) 次多项式（特征多项式）。

### 求解三步法

1. **写特征方程**：\(\det(A - \lambda I) = 0\)
2. **解特征值**：得到 \(\lambda_1, \lambda_2, \dots, \lambda_n\)（实/复，可能重根）
3. **求特征向量**：对每个 \(\lambda_i\)，解齐次方程 \((A - \lambda_i I)\boldsymbol{v} = 0\)，得到一组基

### 完整例子

\[
A = \begin{bmatrix} 4 & -2 \\ 1 & 1 \end{bmatrix}
\]

**Step 1**：
\[
\det \begin{bmatrix} 4-\lambda & -2 \\ 1 & 1-\lambda \end{bmatrix}
= (4-\lambda)(1-\lambda) - (-2)(1)
= \lambda^2 - 5\lambda + 6 = 0
\]

**Step 2**：\(\lambda_1 = 2,\ \lambda_2 = 3\)

**Step 3**：
- \(\lambda_1 = 2\)：\((A-2I)\boldsymbol{v} = \begin{bmatrix} 2 & -2 \\ 1 & -1 \end{bmatrix} \boldsymbol{v} = 0 \Rightarrow \boldsymbol{v}_1 = \begin{bmatrix} 1 \\ 1 \end{bmatrix}\)
- \(\lambda_2 = 3\)：\((A-3I)\boldsymbol{v} = \begin{bmatrix} 1 & -2 \\ 1 & -2 \end{bmatrix} \boldsymbol{v} = 0 \Rightarrow \boldsymbol{v}_2 = \begin{bmatrix} 2 \\ 1 \end{bmatrix}\)

**验证**：\(A \boldsymbol{v}_1 = \begin{bmatrix} 2 \\ 2 \end{bmatrix} = 2\boldsymbol{v}_1\) ✓

### 注意

- 特征向量**不唯一**：任何非零倍数都是特征向量（即整条直线）。NumPy 默认返回**单位长度**的特征向量。
- 同一特征值对应的所有特征向量 + 零向量 = **特征子空间（eigenspace）**。

## 三、关键性质（面试 + 推导工具）

### 代数关系（"迹积律"）

设 \(\lambda_1, \dots, \lambda_n\) 是 \(A\) 的全部特征值（计重数），则：

| 性质 | 公式 | 直觉 |
|------|------|------|
| **迹** | \(\operatorname{tr}(A) = \sum \lambda_i\) | 对角元之和 = 特征值之和 |
| **行列式** | \(\det(A) = \prod \lambda_i\) | 体积缩放因子 = 所有方向缩放的乘积 |
| **可逆性** | \(A\) 可逆 \(\Leftrightarrow\) 所有 \(\lambda_i \neq 0\) | 任一方向被压成 0 → 整体降维 → 不可逆 |
| **\(A^k\)** | 特征值变为 \(\lambda_i^k\)，特征向量不变 | 同一方向重复拉伸 |
| **\(A^{-1}\)**（若存在） | 特征值变为 \(1/\lambda_i\) | 反向缩放 |
| **\(A + cI\)** | 特征值变为 \(\lambda_i + c\) | 各方向再加常数拉伸 |
| **\(A^\top\)** | 与 \(A\) 同特征值（特征向量不同） | 行/列空间对偶 |
| **相似矩阵** | \(P^{-1}AP\) 与 \(A\) 同特征值 | 换基底不改变本质变换 |

> **重要推论**：用迹和行列式可以**反求** 2×2 矩阵的特征值（韦达定理）：
> \(\lambda^2 - \operatorname{tr}(A)\lambda + \det(A) = 0\)

### 代数重数 vs 几何重数

- **代数重数**：\(\lambda\) 在特征多项式中作为根的重数。
- **几何重数**：\(\lambda\) 对应特征子空间的维数（线性无关特征向量个数）。
- 始终满足：**几何重数 ≤ 代数重数**。
- 两者相等 \(\Leftrightarrow\) **可对角化**。

#### 缺陷矩阵（不可对角化）的例子
\[
A = \begin{bmatrix} 2 & 1 \\ 0 & 2 \end{bmatrix}
\]
特征值 \(\lambda = 2\)（代数重数 2），但只有 1 个线性无关特征向量 \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\)（几何重数 1）→ 不可对角化（要走 Jordan 标准形）。

## 四、特征值分解（Eigendecomposition）

### 定义

若 \(A\) 有 \(n\) 个线性无关特征向量 \(\boldsymbol{v}_1, \dots, \boldsymbol{v}_n\)，把它们作为列拼成 \(P\)，对应特征值放在对角矩阵 \(\Lambda\) 上：

\[
P = [\boldsymbol{v}_1 | \boldsymbol{v}_2 | \cdots | \boldsymbol{v}_n], \quad
\Lambda = \operatorname{diag}(\lambda_1, \dots, \lambda_n)
\]

则：

\[
\boxed{A = P \Lambda P^{-1}}
\]

### 三步走的几何意义

把 \(A \boldsymbol{x}\) 写成：
\[
A \boldsymbol{x} = P \Lambda P^{-1} \boldsymbol{x}
\]

读法（**从右到左**）：
1. \(P^{-1} \boldsymbol{x}\)：把 \(\boldsymbol{x}\) **换到特征向量基底**下表达；
2. \(\Lambda (\cdot)\)：在新基底里**沿各坐标轴独立拉伸**；
3. \(P (\cdot)\)：把结果**换回原始基底**。

> **核心洞察**：在合适的基底下，复杂的矩阵变换其实就是"沿各轴独立拉伸"。这就是"对角化"的全部意义。

### 矩阵幂的快速计算

\[
A^k = P \Lambda^k P^{-1}, \qquad \Lambda^k = \operatorname{diag}(\lambda_1^k, \dots, \lambda_n^k)
\]

把 \(O(n^3 k)\) 的矩阵连乘降为 \(O(n^3 + nk)\)。Markov 链的稳态分布、Fibonacci 通项都是这么算的。

### 矩阵函数

只要 \(f\) 可由幂级数定义（\(\exp, \sin, \log, \sqrt{\cdot}\)）：
\[
f(A) = P f(\Lambda) P^{-1} = P \operatorname{diag}(f(\lambda_1), \dots, f(\lambda_n)) P^{-1}
\]
**矩阵指数** \(e^{At}\) 是连续时间动力系统、SDE、神经 ODE 的基础。

## 五、谱定理（实对称矩阵 = AI 的圣杯）

### 定理内容

若 \(A\) 是**实对称矩阵**（\(A = A^\top\)），则：
1. **所有特征值都是实数**
2. **不同特征值对应的特征向量相互正交**
3. **一定可正交对角化**：

\[
\boxed{A = Q \Lambda Q^\top, \quad Q^\top Q = I}
\]

\(Q\) 的列是单位正交特征向量，\(Q^{-1} = Q^\top\)（计算极简）。

### 为什么 AI 这么爱它？

| 场景 | 哪个对称矩阵 | 用途 |
|------|------------|------|
| **PCA** | 协方差矩阵 \(\Sigma = \frac{1}{n} X^\top X\) | 找方差最大方向 |
| **谱聚类** | 图拉普拉斯矩阵 \(L = D - W\) | 把图嵌入低维 |
| **二次型优化** | Hessian \(\nabla^2 f\) | 判断凸/凹/鞍点 |
| **核方法** | Gram 矩阵 \(K_{ij} = k(x_i, x_j)\) | 半正定性 = 合法核 |
| **GNN** | 归一化邻接矩阵 | 图卷积 = 在谱域滤波 |

### 半正定 / 正定（PSD / PD）

实对称矩阵 \(A\) 称为：
- **正定**：所有 \(\lambda_i > 0\) \(\Leftrightarrow \boldsymbol{x}^\top A \boldsymbol{x} > 0,\ \forall \boldsymbol{x} \neq 0\)
- **半正定（PSD）**：所有 \(\lambda_i \geq 0\) \(\Leftrightarrow \boldsymbol{x}^\top A \boldsymbol{x} \geq 0\)

> 协方差矩阵、\(X^\top X\)、Gram 矩阵都**必然 PSD**——这是 PCA 能用、SVM 核能用的根本前提。

## 六、与 SVD 的关系（极易混淆）

### 三句话讲清楚

- **特征值分解**：只对**方阵**且**可对角化**才有；用一组（不一定正交的）基底沿轴拉伸。
- **SVD**：对**任意 \(m \times n\) 矩阵**都成立；分两组**正交基**（输入空间一组，输出空间一组）。
- **联系**：对**实对称半正定矩阵**，SVD 和特征分解**完全一致**（\(U = V = Q\)，\(\Sigma = \Lambda\)）。

### 通过 SVD 求特征结构

\[
A = U \Sigma V^\top
\]

- \(A^\top A = V \Sigma^2 V^\top\)：\(V\) 的列是 \(A^\top A\) 的特征向量，\(\Sigma^2\) 是其特征值。
- \(A A^\top = U \Sigma^2 U^\top\)：\(U\) 的列是 \(A A^\top\) 的特征向量。

> 这就是为什么 PCA 既可以用"协方差 + 特征分解"，也可以用"中心化 \(X\) 直接做 SVD"——两条路殊途同归，**SVD 数值更稳定**，sklearn 内部就是这么实现的。

## 七、AI 中的 5 个高频应用

### 1. PCA（主成分分析）

数据中心化后的协方差矩阵 \(C = \frac{1}{n} X^\top X\)：
- **最大特征值对应的特征向量** = 数据方差最大的方向 = 第一主成分
- 取前 \(k\) 个主成分降维 → **保留信息最多 + 压缩噪声**
- 累计方差贡献率 \(\frac{\sum_{i=1}^{k}\lambda_i}{\sum_{i=1}^{n}\lambda_i}\) 通常取 85%~95%

### 2. 优化中的 Hessian

二阶导数矩阵 \(H = \nabla^2 f\) 的特征值：
- 全部 > 0 → **极小值**（凸）
- 全部 < 0 → 极大值（凹）
- **有正有负 → 鞍点**（深度学习的主要敌人）
- 最大/最小特征值之比 = **条件数**，决定梯度下降的收敛速度

### 3. 协方差矩阵 / 多元高斯

多元高斯 \(\mathcal{N}(\mu, \Sigma)\) 的等概率密度面是椭球：
- **椭球的轴方向** = \(\Sigma\) 的特征向量
- **椭球的半轴长** \(\propto \sqrt{\lambda_i}\)
- 这就是为什么"方差最大方向"等于"主成分方向"。

### 4. PageRank / Markov 链稳态

转移矩阵 \(P\)（列随机）：
- **\(\lambda = 1\)** 必然是其特征值
- 对应的特征向量 = **稳态分布**（PageRank 分数）
- 真实网页规模上亿，特征分解不可行，用**幂迭代**只求最大特征向量。

### 5. 谱图论 / GNN

图拉普拉斯 \(L = D - W\) 的特征值/向量：
- 第二小特征值（**Fiedler 值**）→ 图的"连通强度"
- 对应特征向量 → 谱聚类的最优切分
- 图卷积 GCN 本质是在拉普拉斯特征基下做滤波

## 八、数值计算实战

### NumPy 基本用法

```python
import numpy as np

A = np.array([[4, -2],
              [1,  1]], dtype=float)

# 一般矩阵
eigvals, eigvecs = np.linalg.eig(A)         # 可能返回复数
print("特征值:", eigvals)                    # [3. 2.]
print("特征向量(列):\n", eigvecs)            # 单位长度

# 对称矩阵 —— 更稳更快，保证返回实数
S = (A + A.T) / 2
eigvals, eigvecs = np.linalg.eigh(S)        # eigh 专为 Hermitian/对称
print(eigvals)                               # 已升序排列

# 验证 Av = λv
for i, λ in enumerate(eigvals):
    v = eigvecs[:, i]
    assert np.allclose(S @ v, λ * v)
```

> **重要**：`eig` 返回**列向量**形式的特征向量矩阵，访问第 i 个特征向量是 `eigvecs[:, i]`，不是 `eigvecs[i]`。

### 幂迭代法（Power Iteration）—— 大规模场景

适用于"只想要最大特征值"的场景（PageRank、PCA 第一主成分）。

```python
def power_iteration(A, num_iter=1000, tol=1e-10):
    n = A.shape[0]
    v = np.random.randn(n)
    v /= np.linalg.norm(v)
    for _ in range(num_iter):
        v_new = A @ v
        v_new /= np.linalg.norm(v_new)
        if np.linalg.norm(v_new - v) < tol:
            break
        v = v_new
    λ = v @ A @ v          # Rayleigh 商
    return λ, v
```

**思想**：随机向量被 \(A\) 反复作用后，会逐渐"对齐"到最大特征值方向（前提：\(|\lambda_1| > |\lambda_2|\)）。

**进阶变体**：
- 反幂迭代：求最小特征值（对 \(A^{-1}\) 用幂迭代）
- 移位反幂迭代：求接近某个 \(\sigma\) 的特征值
- Lanczos / Arnoldi：稀疏大矩阵的前 \(k\) 个特征值

### 手写 PCA

```python
def pca(X, k):
    """X: (n_samples, n_features)，返回降维后的 (n_samples, k)"""
    X = X - X.mean(axis=0)                       # 1. 中心化
    C = (X.T @ X) / (len(X) - 1)                 # 2. 协方差矩阵 (对称)
    eigvals, eigvecs = np.linalg.eigh(C)         # 3. 特征分解
    idx = eigvals.argsort()[::-1]                # 4. 降序
    W = eigvecs[:, idx[:k]]                      # 5. 取前 k 个主成分
    return X @ W, eigvals[idx]                    # 投影 + 解释方差
```

### 椭圆可视化（理解"主轴 = 特征向量"）

```python
import matplotlib.pyplot as plt

A = np.array([[3, 1], [1, 2]], dtype=float)
θ = np.linspace(0, 2*np.pi, 200)
circle = np.stack([np.cos(θ), np.sin(θ)])    # 单位圆 (2, 200)
ellipse = A @ circle                          # 变换后变椭圆

eigvals, eigvecs = np.linalg.eigh(A)

plt.plot(*circle, label="单位圆")
plt.plot(*ellipse, label="A 作用后")
for i in range(2):
    v = eigvecs[:, i] * eigvals[i]
    plt.arrow(0, 0, v[0], v[1], width=0.05, color="red")
plt.axis("equal"); plt.legend()
```

椭圆的**长短轴方向就是特征向量**，**长度就是 \(|\lambda|\)**。

## 九、易错点 & 常见误区

| 误区 | 真相 |
|------|------|
| 任意矩阵都能对角化 | ❌ 缺陷矩阵（如 \(\begin{bmatrix}2&1\\0&2\end{bmatrix}\)）不行 |
| 特征值是实数 | ❌ 一般矩阵可能是复数；**实对称矩阵**才一定实 |
| 特征向量唯一 | ❌ 任何非零倍数都行；NumPy 给出单位向量 |
| 旋转矩阵有实特征向量 | ❌ 一般没有（除 0°/180°）；特征值是 \(e^{\pm i\theta}\) |
| `eig` 返回的特征向量是行 | ❌ 是**列**！用 `eigvecs[:, i]` |
| 特征值越大越"重要" | ⚠️ 仅在**对称**或绝对值意义下成立；一般矩阵要看模 \(|\lambda|\) |
| 一般矩阵的 SVD 等于特征分解 | ❌ 仅 PSD 对称矩阵成立 |
| 大矩阵直接调 `eig` | ⚠️ \(O(n^3)\)；只要前几个时用幂迭代 / Lanczos |
