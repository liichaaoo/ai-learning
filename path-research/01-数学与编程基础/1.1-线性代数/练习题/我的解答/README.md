## 推荐文件组织

### 按章节
```
我的解答/
├── ch01/
│   ├── ch01-5.py       # 第 1 章第 5 题的 NumPy 实现
│   ├── ch01-12.py
│   └── ch01-概念题.md
├── ch02/
│   └── ...
└── ...
```

### 或者按题目类型
```
我的解答/
├── notebooks/          # Jupyter Notebook
│   ├── ch01-vectors.ipynb
│   └── ch02-matrices.ipynb
├── scripts/            # 独立 Python 脚本
│   ├── my_dot.py
│   ├── gram_schmidt.py
│   └── my_pca.py
└── notes/              # 手推公式、概念理解
    └── ch05-特征值-我的理解.md
```

## 自查 Checklist

每做完一题问自己：
- [ ] 能不查资料独立讲清楚解题思路吗？
- [ ] 几何直觉上能说通吗？
- [ ] 知道这个知识点将来在哪里用？
- [ ] 写了验证代码吗？
