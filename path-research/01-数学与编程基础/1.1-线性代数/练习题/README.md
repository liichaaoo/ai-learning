## 使用方法

1. **学习 → 动手**：学完一个知识点立即做对应章节题，不要攒到最后
2. **先思考再看答案**：答案在 `答案/` 目录下，做完再对
3. **写代码验证**：每一道计算题都用 NumPy 跑一遍，答案和手算对比
4. **记录解答**：把自己的解答放到 `我的解答/` 目录，格式：`chXX-题X.py` 或 `.md`
