## 📝 概念题

### 6.1 ⭐
**SVD 定义**：对任意 m×n 矩阵 A，
`A = U Σ V^T`
- U：m×m 正交矩阵（左奇异向量）
- Σ：m×n 对角矩阵（奇异值，非负，降序排列）
- V：n×n 正交矩阵（右奇异向量）

问：
- SVD 存不存在是否有限制？（答：对**任意**矩阵都存在，方/非方、奇异/可逆都行）
- 特征值分解要求什么？（方阵 + 可对角化）

### 6.2 ⭐⭐
SVD 与特征值分解的关系：
- (a) `A^T A` 的特征值 = A 奇异值的平方
- (b) `A^T A` 的特征向量 = V 的列
- (c) `A A^T` 的特征向量 = U 的列

为什么？用 `A = UΣV^T` 代入推导。

### 6.3 ⭐⭐
**几何视角**：任意矩阵 A 的作用 = 三步
1. `V^T`：旋转
2. `Σ`：沿坐标轴拉伸
3. `U`：再旋转

用图（单位圆 → 椭圆）说明。

### 6.4 ⭐⭐⭐
**低秩近似 Eckart-Young 定理**：
保留前 k 个最大的奇异值得到 `A_k`，那么 `A_k` 是 A 在 Frobenius 范数意义下的**最优 rank-k 近似**。
- 为什么这个定理重要？（图像压缩、PCA、推荐系统）

## ✍️ 计算题

### 6.5 ⭐⭐
手算 2×2 矩阵的 SVD：`A = [[3, 0], [4, 5]]`

步骤：
1. 计算 `A^T A`
2. 求特征值 λ1, λ2（降序）
3. 奇异值 σ_i = √λ_i
4. 特征向量 → V 的列
5. `u_i = (1/σ_i) A v_i` → U 的列

### 6.6 ⭐⭐
验证：对任意矩阵 A，**rank(A) = 非零奇异值的个数**。
- 用上题矩阵验证
- 再用一个 rank-1 的矩阵（如 `[[1,2],[2,4]]`）验证

### 6.7 ⭐⭐⭐
**Moore-Penrose 伪逆**：`A^+ = V Σ^+ U^T`（`Σ^+` 是 Σ 每个非零元素取倒数后转置）
- 对**非方阵**和**奇异**矩阵都定义良好
- 若 A 可逆，则 `A^+ = A^{-1}`
- 应用：最小二乘 `x = A^+ b`

手算 `A = [[1, 0], [0, 0]]` 的伪逆。

### 6.8 ⭐⭐
用 NumPy 做 SVD 并验证：
```python
A = np.random.randn(5, 3)
U, S, Vt = np.linalg.svd(A, full_matrices=False)
# 验证 U @ diag(S) @ Vt == A
assert np.allclose(U @ np.diag(S) @ Vt, A)
# 验证正交性
assert np.allclose(U.T @ U, np.eye(3))
assert np.allclose(Vt @ Vt.T, np.eye(3))
```

### 6.11 ⭐⭐
**PCA 与 SVD 的等价性**：
- 数据矩阵 X（N 行，D 维），中心化后
- 协方差矩阵 `C = (1/N) X^T X`
- 对 X 直接做 SVD：`X = UΣV^T`
- 问：V 的列 = C 的特征向量？（答：是！证明？）

**好处**：直接对 X 做 SVD 数值上比算协方差矩阵再特征分解更稳定。

**LLM 中的 LoRA**（重要！）：
LoRA 假设模型权重更新 `ΔW` 是**低秩的**：
`ΔW = A · B^T`，其中 A 是 d×r，B 是 r×k，r 远小于 d, k

- 与 SVD 低秩近似的关系？
- 参数量从 `d·k` 降到 `r(d+k)`，若 r=8, d=k=4096，压缩比是多少？
