## 为什么保留这条路径

虽然当前的主线是工程派（`path-engineer`），但这条学院派路径仍然有价值：

1. **完整的学习地图**：未来如果想深入算法或转研究方向，这是已铺好的路
2. **真实积累的内容**：现在已经写好的笔记和练习题（线性代数、监督/无监督学习等）
3. **作为参考资料**：工程派遇到原理问题时，可以来这里查阅
4. **路径选择的备份**：保留另一种可能性，决策不可逆但路径可以转换
