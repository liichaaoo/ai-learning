# 线性回归（Linear Regression）详解

> 经典 ML 的"Hello World"，所有监督学习算法的数学起点。吃透它 = 吃透 MLE、梯度下降、正则化、偏差-方差四大基础概念。

---

## 一、什么是线性回归

**线性回归**：用一条（超）平面去拟合输入 \(x\) 和连续输出 \(y\) 之间的关系。

> 类比：给你一堆 `(房屋面积, 价格)` 的点，在图上画一条直线，让所有点到这条直线的距离"整体最小"——这条直线就是线性回归模型。

### 数学形式

**单变量（一元线性回归）**：
\[
\hat{y} = w x + b
\]

**多变量（多元线性回归）**：
\[
\hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_d x_d + b = \boldsymbol{w}^\top \boldsymbol{x} + b
\]

把偏置 \(b\) 吸收进 \(\boldsymbol{w}\)（给 \(x\) 加一列常数 1），可以写成更干净的矩阵形式：
\[
\hat{\boldsymbol{y}} = X \boldsymbol{w}, \quad X \in \mathbb{R}^{n \times (d+1)}, \ \boldsymbol{w} \in \mathbb{R}^{d+1}
\]

- \(n\)：样本数
- \(d\)：特征数
- \(X\) 的第一列全为 1（对应偏置）

### 三个核心假设（LINE）

| 假设 | 含义 | 违反后果 |
|------|------|---------|
| **L**inearity 线性 | \(y\) 与 \(x\) 是线性关系 | 欠拟合 |
| **I**ndependence 独立 | 样本之间独立 | 标准误估计错 |
| **N**ormality 正态 | 误差 \(\varepsilon \sim \mathcal{N}(0, \sigma^2)\) | MLE 推导不成立 |
| **E**qual variance 同方差 | 误差方差恒定 | 估计不是最优 |

> **关键洞察**：线性回归是"带高斯噪声假设的最大似然估计"——这就是为什么它用平方损失。

## 二、损失函数：最小二乘（MSE）

### 定义

**均方误差（Mean Squared Error）**：
\[
J(\boldsymbol{w}) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 = \frac{1}{n} \| \boldsymbol{y} - X \boldsymbol{w} \|_2^2
\]

有时写成 \(\frac{1}{2n}\)，系数 \(\frac{1}{2}\) 只是为了求导后约掉 2，不影响最优解。

### 为什么用平方？（3 种视角）

#### 视角 1：几何——最小化垂直距离
\(\| \boldsymbol{y} - X\boldsymbol{w} \|_2^2\) 就是点到超平面的**欧氏距离平方和**，是最自然的"拟合好坏"度量。

#### 视角 2：最大似然估计（MLE）
假设误差 \(\varepsilon_i \sim \mathcal{N}(0, \sigma^2)\)，则：
\[
p(y_i \mid x_i; \boldsymbol{w}) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(y_i - \boldsymbol{w}^\top x_i)^2}{2\sigma^2}\right)
\]

对数似然：
\[
\log L(\boldsymbol{w}) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2} \sum_{i=1}^{n}(y_i - \boldsymbol{w}^\top x_i)^2
\]

**最大化 log-likelihood \(\Leftrightarrow\) 最小化平方和**。这就是 MSE 的来源。

#### 视角 3：高斯-马尔可夫定理
在 LINE 假设下，最小二乘估计（OLS）是**最佳线性无偏估计量（BLUE）**：在所有线性无偏估计中，OLS 方差最小。

### 为什么不用 MAE（绝对值）？

| 损失 | 公式 | 优点 | 缺点 |
|------|------|------|------|
| MSE | \((y - \hat{y})^2\) | 可导、有闭式解、符合高斯假设 | 对异常值敏感 |
| MAE | \(\lvert y - \hat{y} \rvert\) | 对异常值稳健 | 0 点不可导、无闭式解 |
| Huber | 小误差用 MSE、大误差用 MAE | 鲁棒 + 可导 | 多一个超参 |

## 三、求解方法

### 方法 1：最小二乘闭式解（Normal Equation）

对 \(J(\boldsymbol{w}) = \| \boldsymbol{y} - X\boldsymbol{w} \|_2^2\) 求导令为 0：

\[
\frac{\partial J}{\partial \boldsymbol{w}} = -2 X^\top (\boldsymbol{y} - X\boldsymbol{w}) = 0
\]

\[
\Rightarrow X^\top X \boldsymbol{w} = X^\top \boldsymbol{y}
\]

\[
\boxed{\boldsymbol{w}^* = (X^\top X)^{-1} X^\top \boldsymbol{y}}
\]

**优点**：一步到位，精确解  
**缺点**：
- 需要 \(X^\top X\) **可逆**（特征线性无关）
- 复杂度 \(O(d^3)\)，**特征数 \(d > 10^4\) 时非常慢**
- 数值不稳定时改用 **SVD**：`np.linalg.lstsq` 或 `sklearn.LinearRegression`（内部用 SVD）

### 方法 2：梯度下降（Gradient Descent）

**梯度**：
\[
\nabla J(\boldsymbol{w}) = \frac{2}{n} X^\top (X\boldsymbol{w} - \boldsymbol{y})
\]

**更新规则**：
\[
\boldsymbol{w} \leftarrow \boldsymbol{w} - \eta \cdot \nabla J(\boldsymbol{w})
\]

| 变体 | 每次用多少样本 | 特点 |
|------|--------------|------|
| Batch GD | 全部 \(n\) 个 | 稳定、慢 |
| Stochastic GD (SGD) | 1 个 | 快、抖动大 |
| Mini-batch GD | 一个 batch（32/64/…） | **实战首选** |

**关键细节**：
- 学习率 \(\eta\) 太大不收敛、太小收敛慢 → 用 **学习率调度** 或 Adam
- **特征必须归一化**（StandardScaler），否则椭圆等高线让梯度下降走"之"字

### 闭式解 vs 梯度下降

| 维度 | 闭式解 | 梯度下降 |
|------|--------|---------|
| 精度 | 精确解 | 近似解 |
| 复杂度 | \(O(d^3 + nd^2)\) | 每轮 \(O(nd)\) |
| 特征数大（\(d > 10^4\)） | ❌ 内存/时间爆炸 | ✅ 可扩展 |
| 特征共线 | ❌ 不可逆 | ✅ 仍可训练 |
| 在线学习 | ❌ 要重算 | ✅ 可增量更新 |

## 四、正则化：岭回归 & Lasso

### 动机：解决过拟合 + 共线性

当特征多、样本少时，\(X^\top X\) 接近奇异，\(\boldsymbol{w}\) 会非常大且不稳定 → **过拟合**。解决思路：在损失里加一项"惩罚大的 \(\boldsymbol{w}\)"。

### 1. 岭回归（Ridge, L2 正则）

\[
J(\boldsymbol{w}) = \| \boldsymbol{y} - X\boldsymbol{w} \|_2^2 + \lambda \|\boldsymbol{w}\|_2^2
\]

**闭式解**：
\[
\boxed{\boldsymbol{w}^* = (X^\top X + \lambda I)^{-1} X^\top \boldsymbol{y}}
\]

加 \(\lambda I\) 后**必然可逆**，解决共线性。

**效果**：把所有权重"均匀地"压小，不会真正归零。

### 2. Lasso（L1 正则）

\[
J(\boldsymbol{w}) = \| \boldsymbol{y} - X\boldsymbol{w} \|_2^2 + \lambda \|\boldsymbol{w}\|_1
\]

**没有闭式解**，需用**坐标下降**或 **近端梯度（ISTA/FISTA）** 求解。

**效果**：部分权重**精确等于 0** → 自动做**特征选择**。

### 3. Elastic Net（L1 + L2）

\[
J = \| \boldsymbol{y} - X\boldsymbol{w} \|_2^2 + \lambda_1 \|\boldsymbol{w}\|_1 + \lambda_2 \|\boldsymbol{w}\|_2^2
\]

兼具稀疏性和稳定性，特征高度相关时比 Lasso 好。

### L1 为什么稀疏？（面试高频）

**几何解释**：
- L2 正则的约束域是**圆**（光滑），与损失等高线相切点**几乎不会落在坐标轴上** → 所有 \(w\) 都非零但很小。
- L1 正则的约束域是**菱形**（有尖角），损失等高线最先碰到的是**顶点**（在坐标轴上）→ 某些 \(w\) 恰好为 0。

```
    L2 (圆)                 L1 (菱形)
      ___                      /\
    /     \                   /  \
   |   ●   |   ← 切点在内部   \  /   ← 切点在顶点 (轴上)
    \_____/                    \/
                              → 对应 w_i = 0
```

**代数解释**：L1 在 \(w=0\) 处的次梯度区间 \([-\lambda, \lambda]\) 足够大，能"压"住小的梯度，使最优解停在 0。

### Ridge vs Lasso 对比

| 维度 | Ridge (L2) | Lasso (L1) |
|------|-----------|-----------|
| 权重压缩 | 压小，不归零 | 部分归零 |
| 特征选择 | ❌ | ✅ |
| 闭式解 | ✅ | ❌（需迭代） |
| 共线性 | 把权重平均分 | 任选一个，其他归零 |
| 先验（贝叶斯） | 高斯先验 | 拉普拉斯先验 |

### 贝叶斯视角（加分项）

在权重 \(\boldsymbol{w}\) 上加先验后做 MAP 估计：
- 高斯先验 \(\boldsymbol{w} \sim \mathcal{N}(0, \sigma_w^2 I)\) → **L2 正则**
- 拉普拉斯先验 \(\boldsymbol{w} \sim \text{Laplace}(0, b)\) → **L1 正则**

## 五、评估指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **MSE** | \(\frac{1}{n}\sum(y_i - \hat{y}_i)^2\) | 最常用，量纲是 \(y\) 的平方 |
| **RMSE** | \(\sqrt{\text{MSE}}\) | 量纲同 \(y\)，可解释 |
| **MAE** | \(\frac{1}{n}\sum \lvert y_i - \hat{y}_i \rvert\) | 对异常值稳健 |
| **\(R^2\)** | \(1 - \frac{\sum(y_i-\hat{y}_i)^2}{\sum(y_i-\bar{y})^2}\) | **决定系数**，越接近 1 越好；可能为负（比瞎猜还差） |
| **Adjusted \(R^2\)** | 考虑特征数的 \(R^2\) | 防止加特征虚高 |
| **MAPE** | \(\frac{1}{n}\sum \lvert \frac{y_i-\hat{y}_i}{y_i} \rvert\) | 百分比误差，\(y=0\) 时爆炸 |

> **实战建议**：报告时同时给 **RMSE + \(R^2\)**——一个看绝对误差，一个看相对拟合度。

## 六、扩展形式

### 1. 多项式回归

加特征工程让线性模型拟合非线性关系：
\[
\hat{y} = w_0 + w_1 x + w_2 x^2 + w_3 x^3 + \dots
\]
本质还是关于 **\(\boldsymbol{w}\) 线性**，所以仍叫线性模型。

```python
from sklearn.preprocessing import PolynomialFeatures
poly = PolynomialFeatures(degree=3, include_bias=False)
X_poly = poly.fit_transform(X)
```

**警惕**：阶数一高就严重过拟合，务必配 Ridge/Lasso。

### 2. 广义线性模型（GLM）

把"高斯 + 恒等链接"推广到其他分布：

| 模型 | 分布 | 链接函数 | 任务 |
|------|------|---------|------|
| 线性回归 | 高斯 | \(g(\mu) = \mu\) | 连续回归 |
| 逻辑回归 | 伯努利 | \(g(\mu) = \log\frac{\mu}{1-\mu}\) | 二分类 |
| 泊松回归 | 泊松 | \(g(\mu) = \log\mu\) | 计数数据 |

所以**逻辑回归就是 GLM 下的"分类版线性回归"**。

### 3. 加权最小二乘（WLS）

样本不等权：
\[
J = \sum_i w_i (y_i - \hat{y}_i)^2
\]
用于**异方差**（不同样本噪声方差不同）场景。

### 4. 局部加权线性回归（LWLR）

对每个预测点 \(x_0\) 单独拟合一个线性模型，离 \(x_0\) 近的样本权重大。—— 思想接近 KNN，是非参数方法。

## 七、常见坑与实战建议

### 1. 特征要不要归一化？

- **闭式解（OLS / Ridge）**：不归一化也能算，但 Ridge 中 \(\lambda\) 会被特征尺度影响 → **推荐归一化**。
- **梯度下降**：**必须归一化**，否则收敛极慢。
- **Lasso**：**必须归一化**，否则大尺度特征不容易被压缩。

### 2. 分类变量怎么处理？
- **One-Hot**：低基数（< 20）
- **Target Encoding**：高基数（城市、用户 ID）
- 记得 `drop_first=True`，避免共线性（虚拟变量陷阱）

### 3. 多重共线性（Multicollinearity）
- **诊断**：方差膨胀因子 **VIF > 10** 即严重共线
- **解决**：Ridge / Lasso / PCA 降维 / 删掉高相关特征

### 4. 异常值
- MSE 对异常值极敏感
- 解决：Huber 损失、RANSAC、先做 IQR 剔除

### 5. 残差分析（Residual Plot）
画 `残差 vs 预测值`，诊断 LINE 假设：
- 残差**随机散布** → 假设成立
- 残差呈**曲线** → 有非线性关系未捕捉（欠拟合）
- 残差**喇叭形** → 异方差（试 log 变换 \(y\)）

## 八、手写代码（面试级）

### 闭式解实现
```python
import numpy as np

class LinearRegressionClosed:
    def fit(self, X, y):
        X = np.c_[np.ones(len(X)), X]                      # 加偏置列
        # w = (X^T X)^(-1) X^T y；数值稳定起见用 lstsq/pinv
        self.w = np.linalg.pinv(X.T @ X) @ X.T @ y
        return self

    def predict(self, X):
        return np.c_[np.ones(len(X)), X] @ self.w
```

### 梯度下降实现
```python
class LinearRegressionGD:
    def __init__(self, lr=0.01, epochs=1000, l2=0.0):
        self.lr, self.epochs, self.l2 = lr, epochs, l2
        self.history = []

    def fit(self, X, y):
        X = np.c_[np.ones(len(X)), X]
        n, d = X.shape
        self.w = np.zeros(d)
        for _ in range(self.epochs):
            y_hat = X @ self.w
            grad = X.T @ (y_hat - y) / n + self.l2 * self.w  # L2 正则（偏置一般不罚）
            self.w -= self.lr * grad
            self.history.append(np.mean((y_hat - y) ** 2))
        return self

    def predict(self, X):
        return np.c_[np.ones(len(X)), X] @ self.w
```

### sklearn 等价调用
```python
from sklearn.linear_model import LinearRegression, Ridge, Lasso, ElasticNet
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

pipe = Pipeline([
    ("scaler", StandardScaler()),
    ("reg", Ridge(alpha=1.0)),           # 换成 Lasso(alpha=0.1) / ElasticNet(...)
])
pipe.fit(X_train, y_train)

```
