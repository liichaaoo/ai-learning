## 这里是什么

本目录存放的是 **v1 学院派学习路径**（`00-学习总览/学习大纲.md`）的阶段性子目录骨架。

没有被实际填充内容，暂时也不会走到。

为了**保持顶层目录清爽、聚焦主线**（v2），它们被集中搬迁到这里。

## 什么时候会重新启用？

以下任一情况发生时，可以考虑把它们搬回顶层：

1. 🔄 **职业方向转向**：从"AI 应用架构师"转向"算法工程师"
2. 📚 **学有余力**：v2 主线已完成，想进一步深挖原理
3. 🎓 **读研/读博**：进入学术路径
4. 🧠 **业务需要**：新工作中遇到底层技术深挖场景

**搬回命令**（供未来参考）：
```bash
cd ai-learning
git mv _legacy/03-阶段3-Transformer与NLP ./
git mv _legacy/04-阶段4-大模型核心技术 ./
# ... 按需搬迁
```

---

## 这样做的理由

- ✅ **不删除**：未来可能会用到，删了心疼
- ✅ **不移走**：文件路径 Git 历史清晰可追溯
- ✅ **集中隐藏**：减少视觉噪音，聚焦 v2 主线
- ✅ **可逆**：需要时一条 `git mv` 就回来了
